About media breakpoints

What are media breakpoints?

Media breakpoints is where you make seperate types of the website for different sizes of the display.
You are able to make it so that the website looks correct on mobile, tablet, and pcs. And not just one.
It is also used to change the way the website looks based on the rotation of the screen so that it isin't for one orientation.

How do develepers chose what breakpoints to use?

They base it first if it is usable on mobile devices and then basied on what kind of website it is
and what types of devices that they perdict the users seeing the website are going to use.

What are some examples of breakpoints?

Some examples are @media only screen and (min-width: x) used to make the website change based on the size of the screen with x being the size of the screen.
Another exapmle is @media only screen and (max-width: x) used to make it only change untill the width is bigger than what x is set to.
Another big example that is used is @media only screen and (orientation: portrait or lanscape) used to change the website based on what orientation the screen is in.
