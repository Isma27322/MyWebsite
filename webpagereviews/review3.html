What are progressive enhancements?

Progressive enhancemen is where you seperate the html, css, and JavaScript code.
It is so that the code can be reused through mutible files.

Why do you need progressive enhancements?

You need progressive enhancements because it maes makeing websites so much more efficient.
It allows you to seperate the html, css, and JavaScript code into three seperate files that will be able to be used in more than one html website.
If you use progressive enhancement you are just able to import the css and JavaScript files into the html file that you want to use it in.
You are only required to write one css and JavaScript file for mutible html files.
Where if you don't use progressive enhancement you be writeing the css and js code inside the html file.
You will be writeing the same code mutible times if you want the same look and functionality.
There will be a lot of repeat code.
